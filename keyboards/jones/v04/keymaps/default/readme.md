# Defalut keymap for Jones

Simple keymap.

## Layers

- Base
- Lower
- Raise
- Adjust

## LED lighting

2 LEDs. (Left side or Right side.)  
For layer indicator.

## Rotary encoder

Supports up to 2 rotary encoders. (Left bottom and Right bottom.)  

## Audio

Supports audio.  
Music mode and Clicky sound available.
